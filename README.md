#  Ranchi Cab Service Website

Modern, SEO-optimized cab booking website built with React, TypeScript, Tailwind CSS, and Vite. Fully responsive with 20+ pages, Schema.org markup, and WhatsApp/phone integration.

![Ranchi Cab Service](https://images.unsplash.com/photo-1449965408869-eaa3f722e40d?w=1200)

##  Quick Start

### Installation & Development
```bash
npm install
npm run dev
# Open http://localhost:5173
```

### Build & Deploy
```bash
npm run build
vercel --prod
```

##  Features

-  **Fully Responsive** - Mobile, tablet, desktop optimized
-  **SEO Perfect** - Schema.org markup, meta tags, sitemap
-  **Fast Performance** - Vite build, code splitting, lazy loading
-  **Type Safe** - Full TypeScript coverage
-  **20+ Pages** - Home, services, routes, fleet, contact, etc.
-  **Booking System** - WhatsApp & phone integration
-  **Animations** - Smooth Framer Motion animations

##  Tech Stack

- **Framework:** React 18 + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + shadcn/ui
- **Animations:** Framer Motion
- **Routing:** React Router v6
- **SEO:** React Helmet Async
- **Icons:** Lucide React
